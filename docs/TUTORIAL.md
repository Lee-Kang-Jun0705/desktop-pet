## 시작하기

### 설치
1. `Desktop Pet.dmg` 파일을 더블클릭
2. `Desktop Pet.app`을 Applications 폴더로 드래그
3. Applications에서 앱 실행

### 첫 실행 시 권한 설정 (macOS)
앱이 마우스 위치를 추적하려면 **접근성 권한**이 필요합니다.

1. 권한 요청 팝업이 나타나면 **"설정 열기"** 클릭
2. 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용
3. **Desktop Pet** 체크박스 활성화
4. 앱 재시작

## 기본 조작

### 🖱️ 펫 드래그
펫을 **좌클릭 + 드래그**하여 원하는 위치로 이동할 수 있습니다.

### 🎯 클릭하여 이동
바탕화면의 아무 곳이나 **클릭**하면 펫이 그 위치로 달려갑니다.

> 💡 펫 영역 내부를 클릭하면 이동하지 않습니다.

### 🔄 크기 조절
펫 위에서 **마우스 휠**을 굴려 크기를 조절합니다.
- 휠 위로: 확대 (최대 200%)
- 휠 아래로: 축소 (최소 30%)

### 👀 자동 행동
펫은 자동으로 다양한 행동을 합니다:
- **대기(idle)**: 가만히 서있기
- **걷기(walk)**: 느리게 배회
- **달리기(run)**: 빠르게 이동
- **공격(attack)**: 마우스가 가까이 오면 공격 모션

## 클릭 통과 모드

화면 상단 중앙에 모드 제어판이 있습니다.

### 모드 종류

| 모드 | 설명 |
|------|------|
| **자동** | 펫 근처에서만 클릭 가능, 그 외 영역은 통과 |
| **ON** | 항상 클릭 통과 (다른 앱 방해 없음) |
| **OFF** | 항상 클릭 가능 (펫과 상호작용 가능) |

### 모드별 사용 시나리오

**자동 모드 (권장)**
- 일반적인 사용에 적합
- 펫과 상호작용하고 싶을 때 펫 근처로 마우스 이동
- 다른 작업 시 펫이 방해하지 않음

**ON 모드 (클릭 통과 고정)**
- 펫을 구경만 하고 싶을 때
- 작업에 집중하고 싶을 때
- 바탕화면 클릭으로 펫 이동 가능

**OFF 모드 (클릭 가능 고정)**
- 펫과 적극적으로 상호작용하고 싶을 때
- 드래그, 컨텍스트 메뉴 등 자유롭게 사용

## 컨텍스트 메뉴

펫을 **우클릭**하면 컨텍스트 메뉴가 나타납니다.

### 메뉴 항목

| 항목 | 설명 |
|------|------|
| **펫 이름** | Stone Guardian |
| **크기 조절** | 슬라이더로 30%~200% 조절 |
| **공격!** | 공격 모션 실행 |
| **달려!** | 랜덤 위치로 달리기 |

## 시스템 트레이

메뉴바(macOS) 또는 시스템 트레이(Windows)에 펫 아이콘이 표시됩니다.

### 트레이 메뉴

아이콘을 **우클릭**하면 메뉴가 나타납니다:

| 항목 | 설명 |
|------|------|
| **자동 모드** | 클릭 통과 자동 전환 |
| **클릭 통과 ON** | 클릭 통과 고정 |
| **클릭 통과 OFF** | 클릭 가능 고정 |
| **종료** | 앱 종료 |

---

## 펫 행동 패턴

### 마우스 반응
펫은 마우스 커서를 인식하고 반응합니다:

| 거리 | 행동 |
|------|------|
| **90px 이내** | 공격 모션 (15% 확률) 또는 대기 |
| **90~260px** | 걸어서 따라감 |
| **260~520px** | 뛰어서 따라감 |
| **520px 이상** | 자유 배회 |

### 자유 배회
마우스가 멀리 있을 때 펫은 스스로 행동을 결정합니다:
- 35% 확률: 대기
- 35% 확률: 걷기
- 20% 확률: 뛰기
- 10% 확률: 공격 모션

## 문제 해결

### 펫이 마우스를 따라오지 않아요
1. 접근성 권한 확인 (macOS)
2. 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용
3. Desktop Pet 체크 후 앱 재시작

### 다른 앱을 클릭할 수 없어요
1. 클릭 통과 모드를 **자동** 또는 **ON**으로 변경
2. 시스템 트레이에서 모드 변경 가능

### 펫이 화면 밖으로 나갔어요
1. 앱 재시작 시 화면 중앙에 다시 나타남
2. 시스템 트레이 → 종료 후 재실행

### 애니메이션이 끊겨요
1. 다른 앱의 GPU 사용량 확인
2. 펫 크기를 줄여보세요 (크기가 클수록 리소스 사용 증가)
